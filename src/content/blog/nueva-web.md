# Una nueva web construida con Astro, React y Markdown

La nueva web de **Superhéroes INC** no solo inaugura una etapa de expansión del lore y de organización del contenido. También representa un salto técnico importante.  
Hemos optado por una arquitectura moderna, rápida y eficiente basada en **Astro**, **React** y **Markdown**, que garantiza flexibilidad en el desarrollo y una experiencia fluida para los visitantes.

Esta decisión responde al objetivo principal del proyecto:  
crear una plataforma ligera, fácil de actualizar y pensada para crecer al ritmo del universo de Superhéroes INC.
## Un sitio rápido, moderno y modular gracias a Astro

**Astro** se ha convertido en el motor principal de la web.  
Este framework nos permite:

- generar páginas estáticas ultrarrápidas,  
- optimizar el rendimiento sin esfuerzo,  
- cargar React solo donde es realmente necesario,  
- y mantener un entorno de desarrollo limpio y escalable.

Astro es ideal para un proyecto donde conviven cientos de páginas, fichas de lore y documentación estructurada.
## Componentes interactivos con React

Aunque la mayor parte del sitio funciona como contenido estático optimizado, algunos elementos requieren interactividad y dinamismo.  
Para ello, hemos integrado **React** en puntos clave de la web:

- menús avanzados,  
- listados de búsqueda,  
- navegación interna filtrada,  
- componentes reutilizables que convierten la web en una herramienta práctica para jugadores y guionistas.

La combinación de Astro + React nos da lo mejor de ambos mundos:  
velocidad y estabilidad, junto con interactividad moderna.
## Markdown como base documental del lore

Uno de los mayores retos del proyecto era organizar un universo enorme y en expansión constante.  
La solución ha sido clara: **Markdown**.

Todos los contenidos, organizaciones, localizaciones, personajes, eventos, líneas temporales, se almacenan en formato Markdown, lo que permite:

- actualizar fichas rápidamente,  
- mantener un estilo uniforme,  
- vincular documentos con facilidad,  
- y ofrecer al lector un contenido limpio y legible.

Gracias a Markdown, la web funciona como un **repositorio vivo** del canon de Superhéroes INC.
## Una arquitectura preparada para crecer

El diseño técnico de la web facilita que nuevos contenidos se incorporen de manera continua:

- añadir una nueva organización es crear un archivo `.md`,  
- actualizar una localización es tan sencillo como editar una sección,  
- introducir nuevas categorías es inmediato,  
- y garantizar la coherencia del universo es más fácil que nunca.

Esta estructura permite que autores y guionistas trabajen en paralelo sin perder cohesión.

Bienvenidos a la nueva infraestructura del multiverso.
